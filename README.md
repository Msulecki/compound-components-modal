## Modal component
Modal functionality made with help of React Compound Components.

[Live version on Gitpages](https://msulecki.github.io/compound-components-modal/)

### Installation
Clone this repo:
```
git clone git@github.com:Squarespace/base-template-npm.git
```
Instal dependencies:
```
npm install
```
Run local server:
```
npm start
```
